# 智能猫咪健康监护助手（单日检测版）
基于 PyQt5 开发的猫咪健康检测工具，内置分级预警智能体，支持添加/修改猫咪信息、单日健康记录录入、智能健康决策。

## 功能清单
1. 添加猫咪基础信息
2. 修改猫咪信息
3. 录入单日健康记录（饮食量/排便次数/精神状态）
4. 单日健康智能检测（汇总多维度异常，分级预警）

## 环境要求
- Python 3.8+
- 依赖：PyQt5
